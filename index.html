Chargement des membres…

Copiez ces règles dans Firebase Console → Firestore → Rules pour sécuriser votre base.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    function isAdmin() {
      return request.auth != null && request.auth.token.admin == true;
    }
    function isAuthenticated() {
      return request.auth != null;
    }

    // Spécialités : lecture publique, écriture admin seulement
    match /specialites/{docId} {
      allow read: if true;
      allow write: if isAdmin();
    }

    // Fiches : lecture publique, écriture admin seulement
    match /fiches_detaillees/{docId} {
      allow read: if true;
      allow write: if isAdmin();
    }

    // Users : chacun peut lire/créer son propre profil
    match /users/{userId} {
      allow read: if request.auth.uid == userId || isAdmin();
      allow create: if isAuthenticated() && request.auth.uid == userId;
      allow update: if isAdmin();
      allow delete: if isAdmin();
    }
  }
}
Important : le rôle admin doit être attribué via Firebase Admin SDK avec des custom claims. Le champ users/{uid}.role sert à l’affichage, mais la vraie suppression est contrôlée par la règle request.auth.token.admin == true.
